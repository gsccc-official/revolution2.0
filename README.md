## 🎭 About the Event

</div>

**Revolution 2.0** is the flagship national cultural event of the **Govt. Science College Cultural Club (GSCCC)**, celebrating art, identity, and expression. Students from across Bangladesh compete, perform, and inspire across 8 creative segments.

This repository is the **official registration portal** — participants browse segments, register online, pay via bKash, and receive an instant invoice. Everything runs from a single link, on any device.

| Layer | What it does |
|-------|-------------|
| **Frontend** | Pure HTML, CSS, Vanilla JS — zero frameworks, zero build steps |
| **Database** | Supabase (PostgreSQL) — stores all registrations securely |
| **Storage** | Supabase Storage — holds bKash payment screenshots |
| **Auth** | Supabase Auth — protects the admin dashboard |
| **Hosting** | GitHub Pages — free, fast, auto-deploys on every push |
| **Invoice** | html2canvas — generates downloadable receipts client-side |
| **Uptime** | GitHub Actions — pings Supabase every 4 days to prevent sleep |
